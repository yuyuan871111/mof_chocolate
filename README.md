# Project: mof_chocolate 
## 起源
這是2018年由四位大學部同學（謝文瑜、楊淯元、楊雅惠、周信呈）共同在暑期製作的一個project。由國立陽明大學（現國立陽明交通大學）生物醫學資訊研究所楊永正老師指導，並由生物資訊組各位學長姐協助，在此特別感謝蔡博、james、琦瑋、薰文、鴻青的幫助，讓我們能順利完成整個project。
